Highlighting

Create rules based on your list titles, and apply the effects you want. For example, you could turn a list red if its title is “Things to do today”. Or you could make it faded if the title contains the words “done” or “trash”.

Rules

Rule List title Colour More options
Notes
Text matching
All text matching is case-insensitive, e.g. TODO is the same as todo. Only whole words are matched, e.g. “doing” will only match the whole word “doing”. It will not match “undoing” or “#doing”.
Order
Rules are applied in the order they appear above, from top to bottom. The first matching rule will be applied. Once a rule matches a list, processing stops. You can reorder the rules by dragging them using the left hand side.
List title is
This field should contain entire titles. The rule will be applied if the entire title text matches one of these titles.
List title contains
This field should contain parts of titles. The rule will be applied if part of the title text matches, i.e. if the title text contains this. This is useful if you want to match lists with similar names, e.g. “Things to do”, “To do list”, “Things to do this week”.
More options: strikethrough
This option puts a line through all text on the list
More options: greyscale
This option turns all colour on lists to grey
More options: other lists
When a list matches a rule with the “Other list” options turned on, List Highlighter will look for any lists on the board that do not match any rules. It will then apply the rule you specify to these lists. For example, you could have “To do” lists highlighted red, then apply a “Low importance” rule to the other lists on the board.

Other options

Hide hashtags in list headers

With apologies, it has not been possible to restore this option after Trello’s latest update.

Change text colour to contrast with background

If this is enabled, list titles and heading cards will change colour to ensure good contrast between text and background colour.

Default: on

Un-dim dimmed lists on mouse over

If you’ve made lists less visible (more transparent) as part of a highlighting rule, switching this setting on will un-dim them when the mouse cursor hovers over.

Default: on
