Πολιτική απορρήτου για την Προστασία περιήγησης από την F-Secure

Σεπτέμβριος 2022

Εν συντομία

Η Προστασία περιήγησης από την F-Secure είναι μια επέκταση για την προστασία της κρυπτογραφημένης περιήγησης με τα προϊόντα ασφάλειας της F-Secure.

Τα προϊόντα ασφάλειας της F-Secure απαιτούν αυτήν την επέκταση προγράμματος περιήγησης για την προστασία της περιήγησης στο διαδίκτυο, των διαδικτυακών τραπεζικών συναλλαγών, των αγορών σας και για να εμφανίζουν πληροφορίες ασφάλειας κατά την περιήγησή σας στο διαδίκτυο.

Συνοπτικά, η Προστασία περιήγησης από την F-Secure και το απόρρητο:
  • συλλέγουμε ανώνυμα δεδομένα ασφάλειας για την προστασία σας και της συσκευής σας,
  • η συλλογή των δεδομένων δεν εστιάζει σε εσάς αλλά στη συσκευή σας και την υπηρεσία μας.

Αναλυτικά

Ορολογία

  • Η Προστασία περιήγησης από την F-Secure είναι η επέκταση του προγράμματος περιήγησης Firefox για την οποία ισχύει αυτή η πολιτική ιδιωτικού απορρήτου.
  • Το προϊόν ασφάλειας F-Secure αναφέρεται σε οποιοδήποτε προϊόν ασφάλειας που παρέχεται από την F-Secure Corporation, το οποίο συνεργάζεται με την επέκταση Προστασία περιήγησης από την F-Secure. Το όνομα αυτών των προϊόντων ενδέχεται να μην περιέχει την επωνυμία "F-Secure". Σε αυτήν την περίπτωση, μπορούν να αναγνωριστούν από το λογότυπο "by F-Secure".
  • Το F-Secure Security Cloud είναι ένα σύστημα cloud της F-Secure για την ανάλυση απειλών στον κυβερνοχώρο. Χάρη στο Security Cloud, η F-Secure διατηρεί μια ενημερωμένη επισκόπηση του παγκόσμιου τοπίου απειλών και προστατεύει τους πελάτες της από νέες απειλές, μόλις αυτές εντοπιστούν.

Δεδομένα που συλλέγονται από την Προστασία περιήγησης της F-Secure

Ελέγχουμε τις διευθύνσεις URL (διευθύνσεις ιστοσελίδων) των ιστοσελίδων που ανοίγετε, τις διευθύνσεις URL στις οποίες γίνεται παραπομπή από τις ιστοσελίδες που ανοίγετε και παρακολουθούμε τις καρτέλες προγράμματος περιήγησης που ανοίγετε και κλείνετε. Αυτές οι πληροφορίες μεταβιβάζονται στο προϊόν ασφάλειας F-Secure που έχετε εγκαταστήσει στον υπολογιστή σας. Το προϊόν ασφάλειας στέλνει ερωτήματα σχετικά με τις διευθύνσεις URL στο F-Secure Security Cloud, ώστε να λάβει αξιολογήσεις ασφάλειας για τις συγκεκριμένες διευθύνσεις. Στη συνέχεια, ανάλογα με τις αξιολογήσεις ασφάλειας που λαμβάνονται, η Προστασία περιήγησης της F-Secure αποκλείει την πρόσβαση στις ιστοσελίδες που αξιολογούνται ως επιβλαβείς, ύποπτες, παράνομες ή (εφόσον το προϊόν ασφάλειας έχει διαμορφωθεί) ακατάλληλες για παιδιά. Η παρακολούθηση των καρτελών που ανοίγετε και κλείνετε περιορίζεται στον υπολογιστή σας με σκοπό την παρακολούθηση των προστατευμένων περιόδων λειτουργίας τραπεζικών συναλλαγών σας. Δεν προωθούμε πληροφορίες σχετικά με τις καρτέλες του προγράμματος περιήγησης στο F-Secure Security Cloud.

Αν το εγκατεστημένο προϊόν ασφάλειας έχει διαμορφωθεί κατάλληλα, Η Προστασία περιήγησης της F-Secure τροποποιεί τα ερωτήματα αναζήτησης στο διαδίκτυο (για παράδειγμα, μέσω Google ή Bing) ώστε να επιβάλει την πραγματοποίηση των αναζητήσεων σε λειτουργία "Ασφαλούς αναζήτησης", αποκλείοντας σεξουαλικό περιεχόμενο όπως πορνογραφία, βία και αιματοχυσία από τα αποτελέσματα αναζήτησης.

Όταν κάνετε αναζήτηση στο διαδίκτυο χρησιμοποιώντας το Google, το Bing και άλλες μηχανές αναζήτησης, ελέγχουμε τις διευθύνσεις URL των αποτελεσμάτων αναζήτησης και, όπως αναφέρεται παραπάνω, λαμβάνουμε αξιολογήσεις ασφάλειας από το F-Secure Security Cloud. Στη συνέχεια, τροποποιούμε τη σελίδα αποτελεσμάτων αναζήτησης, εισάγοντας "εικονίδια αξιολόγησης" δίπλα στα αποτελέσματα αναζήτησης, ώστε να μπορείτε να δείτε ποιες συνδέσεις στα αποτελέσματα αναζήτησης είναι ασφαλείς, επιβλαβείς, ύποπτες, παράνομες ή ακατάλληλες για παιδιά.

Η Προστασία περιήγησης από την F-Secure παρακολουθεί και συλλέγει μόνο τις διευθύνσεις URL που ανοίγονται και αυτές στις οποίες γίνεται παραπομπή από τις ιστοσελίδες που ανοίγετε. Δεν πραγματοποιούμε ανάγνωση, ούτε τροποποιούμε οποιοδήποτε άλλο τμήμα του περιεχομένου των ιστοσελίδων που ανοίγετε, εκτός από τις διευθύνσεις URL που περιέχονται στις ιστοσελίδες και τα αποτελέσματα αναζήτησης, με σκοπό την εισαγωγή "εικονιδίων αξιολόγησης" όπως αναφέρεται παραπάνω.

Οι διευθύνσεις URL που ανοίγονται ή που περιέχονται στις ιστοσελίδες που ανοίγετε προωθούνται ανώνυμα στο F-Secure Security Cloud (για τους σκοπούς που αναφέρονται παραπάνω), χωρίς πληροφορίες που θα μπορούσαν να σας ταυτοποιήσουν ως το μέρος που ανοίγει τις ιστοσελίδες

Τα δικαιώματά σας

Τα δεδομένα που έχουμε για εσάς σάς ανήκουν. Συγκεκριμένα, έχετε τα παρακάτω δικαιώματα για τα δεδομένα προσωπικού χαρακτήρα που διατηρούμε για εσάς:

  • Πρόσβαση και διόρθωση. Έχετε το δικαίωμα να μας ρωτήσετε ποια προσωπικά δεδομένα έχουμε για εσάς και, στο πλαίσιο αυτό, να λάβετε ένα αντίγραφο των δεδομένων που μπορούμε να προσδιορίσουμε ότι σας αφορούν. Σε περίπτωση που διαπιστώσετε ότι υπάρχουν σφάλματα (π.χ. πληροφορίες που δεν ισχύουν πλέον) σε αυτά τα δεδομένα, σας προτρέπουμε να επικοινωνήσετε με το τμήμα εξυπηρέτησης πελατών της εταιρείας μας, για να επιλύσετε το θέμα. Ορισμένες από τις πύλες εξυπηρέτησης που διαθέτουμε σάς δίνουν τη δυνατότητα να ενημερώνετε τα στοιχεία σας ως πελάτη. Σε αυτήν την περίπτωση, θα πρέπει να ενημερώνετε τα στοιχεία σας εφόσον έχουν γίνει αλλαγές στα προσωπικά δεδομένα, π.χ. αλλαγή διεύθυνσης ή email. Αν δεν μπορείτε να αλλάξετε μόνοι σας τα στοιχεία, μπορείτε να μας ενημερώσετε για τις απαιτούμενες αλλαγές.
  • Ένσταση. Έχετε το δικαίωμα να εναντιωθείτε σε συγκεκριμένες μορφές επεξεργασίας των δεδομένων προσωπικού χαρακτήρα που σας αφορούν, όπως μεταξύ άλλων στην επεξεργασία αυτών των δεδομένων για διαφημιστικούς σκοπούς ή για την υποστήριξη δικού μας έννομου συμφέροντος. Στη δεύτερη περίπτωση, η εναντίωσή σας θα πρέπει να στηρίζεται σε έγκυρη νομική βάση.
  • Δικαίωμα στη λήθη. Έχετε, επίσης, το δικαίωμα να μας ζητήσετε να πάψουμε να αποθηκεύουμε και να διαγράψουμε τα δεδομένα σας προσωπικού χαρακτήρα. Σε αυτήν την περίπτωση, το αίτημά σας θα πρέπει να στηρίζεται σε έγκυρη νομική βάση.
  • Φορητότητα. Έχετε, επίσης, το δικαίωμα να ζητήσετε τα προσωπικά δεδομένα που έχετε παράσχει οι ίδιοι στο πλαίσιο σύμβασης ή κατόπιν της συγκατάθεσής σας. Μπορείτε να ζητήσετε να λάβετε αυτά τα δεδομένα σε δομημένη, συνήθη και μηχανικά αναγνώσιμη μορφή, καθώς επίσης και να ζητήσετε τη διαβίβασή τους σε άλλον υπεύθυνο επεξεργασίας δεδομένων, εφόσον αυτό είναι τεχνικά εφικτό.
  • Απόσυρση συγκατάθεσης. Στις περιπτώσεις όπου η επεξεργασία βασίζεται στη συγκατάθεσή σας, έχετε το δικαίωμα να αποσύρετε αυτή τη συγκατάθεση οποιαδήποτε στιγμή, κάνοντας χρήση των σχετικών ρυθμίσεων. Σε ό,τι αφορά τα ταυτοποιήσιμα δεδομένα στατιστικών υπηρεσιών, μπορείτε να βρείτε τις σχετικές ρυθμίσεις στο περιβάλλον εργασίας. Επίσης, έχετε το δικαίωμα να σταματήσετε να λαμβάνετε διαφημιστικό υλικό μέσω του κέντρου προτιμήσεων στο οποίο μπορείτε να αποκτήσετε πρόσβαση επιλέγοντας τον σχετικό σύνδεσμο.
  • Περιορισμός. Εφόσον διαπιστώσετε ότι τα δεδομένα που έχουμε για εσάς είναι εσφαλμένα ή ότι δεν έχουμε κανένα νόμιμο δικαίωμα να κάνουμε χρήση αυτών των δεδομένων, μπορείτε να μας ζητήσετε να διακόψουμε κάθε περαιτέρω επεξεργασία των προσωπικών δεδομένων σας και απλώς να τα διατηρήσουμε μέχρι να επιλυθεί το θέμα.

Μπορείτε να διεκδικείτε τα δικαιώματά σας μέσω της εξυπηρέτησης πελατών. Οι σύνδεσμοι επικοινωνίας βρίσκονται στην ενότητα "Στοιχεία επικοινωνίας".

Πρέπει να σημειωθεί ότι ενδέχεται να υπάρξουν περιπτώσεις όπου οι υποχρεώσεις μας περί εμπιστευτικότητας, το δικαίωμά μας στην επαγγελματική εχεμύθεια ή/και οι υποχρεώσεις μας για την παροχή των υπηρεσιών μας (π.χ. προς τον εργοδότη σας) μπορεί να μην επιτρέπουν την κοινοποίηση ή τη διαγραφή των προσωπικών δεδομένων που σας αφορούν ή να εμποδίζουν με άλλον τρόπο την άσκηση των δικαιωμάτων σας. Τα παραπάνω δικαιώματά σας εξαρτώνται επίσης από νομικά θέματα βάσει των οποίων επεξεργαζόμαστε τα δικά σας προσωπικά δεδομένα.

Εάν έχετε παράπονα για τον τρόπο με τον οποίο επεξεργαζόμαστε τα προσωπικά δεδομένα σας ή εάν χρειάζεστε περισσότερες πληροφορίες, μπορείτε να επικοινωνείτε μαζί μας ανά πάσα στιγμή. Εάν πιστεύετε ότι δεν σας εξασφαλίζουμε τα νόμιμα δικαιώματά σας, μπορείτε να υποβάλετε καταγγελία σε κάποια αρμόδια αρχή. Στην πλειοψηφία των περιπτώσεων, αυτή η αρχή είναι ο Συνήγορος Προστασίας Δεδομένων της Φινλανδίας (www.tietosuoja.fi).
