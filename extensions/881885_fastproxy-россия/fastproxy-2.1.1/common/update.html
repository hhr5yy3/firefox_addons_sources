«FastProxy - обход блокировки сайтов» обновился!

Дополнительные функции в меню настроек

  • Настройка собственных списков сайтов для проксирования
    Добавлена возможность создавать свои списки сайтов для проксирования. Доступны следующие опции:

    «Не проксировать домены из списка» - создать список доменов которые не будут проксироваться через расширение.
    «Проксировать только домены из списка» - создать список доменов которые будут проксироваться через расширение. Все остальные домены будут работать напрямую.
    «Добавить домены для проксирования» - добавить к списку сайтов которые проксирует расширение свои домены.
    «Проксировать весь трафик» - проксировать весь трафик через прокси - работает как VPN.
