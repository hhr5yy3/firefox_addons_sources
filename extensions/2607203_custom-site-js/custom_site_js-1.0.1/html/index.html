Custom Site JavaScript is a chrome extension that lets you define javascript which will automatically be executed on a site as it appears in your browser.

You can Add, Edit, or Delete CSJ items. In the edit you can also mark a CSJ item to be skipped so that you can disable it without deleting it.

Add/Edit:
  • Name is the CSJ item in your list (limit 30 chars).
  • Skip, when checked, disables the item without needing to delete it.
  • URL is the string the site url is tested with. This url string can be created using * to indicate a zero or more char wild card, or ** for a one or more char wild card. Note: Chrome tacks a / on the end of a url that only has the domain name.
  • Regex, when checked, allows URL to be a full regex test string. Example: .*google\.com\/search.+ If you need to use flags, you can format it like this: /MATCH/FLAGS. Example: /.*google\.com\/search.+/i
  • Script is the javascript code you want to have run. The total size that can be stored is about 5 megabytes.
