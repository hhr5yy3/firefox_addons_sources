XMLDocReader-додаток для браузерiв Firefox та Chrome для автоматизації заповнення полiв даними з файлу обмiну (XML файлу) в особистому кабiнетi "Електроннi сервiси Держгеокадастру" та набором додаткових опцій для зручності роботи. Додаток автоматично визначає сторiнку для аналізу, що вiдображається у верхньому кутi значка додатку та заповнює необхідні поля інформацією з обмінного файлу.
