F-Secure 개인 정보 보호 정책에 의한 브라우징 보호

2022년 9월

개요

F-Secure의 안전 브라우저는 F-Secure의 보안 제품으로 암호화된 브라우징을 보호하기 위한 확장입니다.

F-Secure의 보안 제품을 사용하려면 웹 브라우징, 온라인 뱅킹 및 쇼핑을 보호하고 인터넷을 탐색하는 동안 보안 정보를 표시할 수 있는 이 브라우저 확장이 필요합니다.

F-Secure 개인 정보 보호 정책에 의한 브라우징 보호
  • 익명의 보안 데이터를 수집하여 장치를 보호합니다.
  • 데이터 수집은 여러분이 아닌 여러분의 장치와 F-Secure의 서비스에 목적이 있습니다.

전문

용어

  • F-Secure의 안전 브라우저는 본 개인 정보 보호 정책이 적용되는 Firefox 브라우저 확장입니다.
  • F-Secure 보안 제품이란 F-Secure에서 제공하며 F-Secure의 안전 브라우저 확장과 함께 작동하는 모든 보안 제품을 말합니다. 그러한 제품에는 "F-Secure"가 아닌 다른 브랜드 이름이 붙을 수 있으며, 그러한 경우 "by F-Secure" 로고로 F-Secure 보안 제품임을 식별할 수 있습니다.
  • F-Secure Security Cloud는 F-Secure에서 운영하는 클라우드 기반 사이버 위협 분석 시스템입니다. F-Secure는 Security Cloud를 통해 글로벌 위협 환경에 대한 최신 개요를 유지하고 새로운 위협이 처음 발견되는 순간부터 고객을 그러한 위협으로부터 보호할 수 있습니다.

F-Secure의 안전 브라우저가 수집한 데이터

당사는 귀하가 연 웹 페이지의 URL(웹 페이지 주소)과 귀하가 연 웹 페이지에서 참조한 URL을 확인하고, 귀하가 열고 닫은 브라우저 탭을 추적합니다. 이 정보는 컴퓨터에 설치된 F-Secure 보안 제품으로 전송됩니다. 해당 보안 제품은 URL에 대한 쿼리를 F-Secure Security Cloud에 전송하여 해당 URL의 보안 등급을 파악합니다. 그리고 파악한 보안 등급에 따라 F-Secure의 안전 브라우저가 유해하거나 의심스럽거나 불법이거나 (보안 제품의 구성에 따라) 아동에게 적절하지 않은 등급을 받은 웹 페이지에 대한 액세스를 차단합니다. 열거나 닫은 브라우저 탭 추적은 귀하의 컴퓨터에서 보안 뱅킹 세션을 추적하는 목적으로만 사용됩니다. 당사는 귀하의 브라우저 탭에 대한 정보를 F-Secure Security Cloud로 전달하지 않습니다.

설치된 보안 제품의 구성에 따라, F-Secure의 안전 브라우저가 인터넷 검색 쿼리(예: Google 또는 Bing 사용)를 변경하여 "안전 검색" 모드에서만 검색하도록 강제할 수도 있습니다. 그러면 음란물처럼 성적으로 노골적인 콘텐츠, 폭력적이거나 잔인한 콘텐츠는 검색 결과에서 차단됩니다.

Google, Bing 및 기타 검색 엔진으로 인터넷 검색을 할 때, 당사는 검색 결과로 나온 URL을 확인하고 위에 설명한 것처럼 F-Secure Security Cloud에 그러한 URL의 보안 등급을 쿼리합니다. 그리고 검색 결과 옆에 "등급 아이콘"을 삽입하여 검색 결과 페이지를 변경합니다. 그러면 귀하는 검색 결과에서 어떤 링크가 안전하거나 유해하거나 의심스럽거나 불법이거나 아동에게 적절하지 않은지 볼 수 있습니다.

F-Secure의 안전 브라우저는 귀하가 연 URL과 귀하가 연 웹 페이지에서 참조하는 URL만 모니터링하고 수집합니다. 당사는 귀하가 연 웹 페이지에서 거기에 담긴 URL 및 검색 결과 외의 내용을 읽거나 변경하지 않으며, 위에 설명한 것처럼 검색 결과에 "등급 아이콘"을 삽입하는 목적으로만 그렇게 합니다.

귀하가 연 URL 또는 귀하가 연 웹 페이지에 담긴 URL은 익명화되어 (위에 설명한 목적으로) F-Secure Security Cloud에 전송되며, 당사는 해당 웹 페이지를 연 사람이 누구인지 알 수 없습니다.

귀하의 권한

귀하는 귀하에 대해 당사에서 보유한 데이터에 대한 권한을 갖습니다. 특히, 귀하에 대해 당사에서 보유한 개인 데이터에 대해 다음 권한을 갖습니다.

  • 액세스 및 수정. 귀하는 귀하에 대해 당사에서 보유한 개인 데이터가 무엇인지 질문하고 해당 맥락에서 당사에서 귀하에 관하여 식별할 수 있는 데이터의 사본을 받을 권리가 있습니다. 그러한 데이터에서 오류(예: 사용하지 않는 데이터)를 발견하는 경우 고객 서비스 팀에 연락하여 문제를 해결하는 것이 좋습니다. 당사의 일부 서비스 포털을 통해 고객 정보를 업데이트할 수 있습니다. 그와 같이, 주소 또는 이메일 주소의 변경과 같은 개인 데이터의 모든 변경사항을 업데이트해야 합니다. 직접 변경사항을 업데이트할 수 없으면 필요한 변경사항에 대해 당사에 알릴 수 있습니다.
  • 반대. 귀하는 예컨대 마케팅 목적으로 개인 데이터를 처리하거나 합법적 이해에 따라 달리 귀하의 데이터를 처리하는 경우, 개인 데이터의 특정한 처리에 반대할 권리가 있습니다. 후자의 경우 법적으로 유효한 반대의 근거를 규명해야 합니다.
  • 잊혀질 권리. 귀하는 당사에 개인 데이터의 저장을 중단하고 지우라고 요청할 권리도 있습니다. 이 경우 요청에 대한 법적으로 유효한 근거를 설정해야 합니다.
  • 휴대성. 귀하는 또한 계약 또는 동의에 따라 귀하가 직접 제공한 개인 데이터에 대해 질문할 권리도 있습니다. 구조화되고 일반적으로 사용되고 기계 판독이 가능한 형식으로 데이터를 요청하고 추가로 기술적으로 실행 가능한 경우 데이터가 다른 관리자에게 전송되도록 요청할 수 있습니다.
  • 동의 철회. 처리가 귀하의 동의에 따라 이루어지는 경우 귀하는 관련 설정을 통해 언제든지 동의를 철회할 권리가 있습니다. 식별 가능한 서비스 분석 데이터의 경우 귀하는 서비스 사용자 인터페이스에서 설정을 찾을 수 있습니다. 귀하는 또한 링크를 통해 액세스할 수 있는 기본 설정 센터를 통해 마케팅 통신을 옵트 아웃할 권리도 보유합니다.
  • 제한. 귀하가 당사가 보유하고 있는 귀하와 관련된 데이터가 부정확하거나 당사에서 해당 데이터를 사용할 법적 권리가 없다고 언급하는 경우 귀하는 당사에 귀하의 개인 데이터를 더 이상 처리하지 않고 문제가 해결될 때까지 보관만 하도록 요청할 수 있습니다.

귀하는 고객 서비스 팀을 통해 귀하의 권리를 행사할 수 있습니다. 당사에 연락하는 링크는 "연락처 정보" 섹션에서 확인할 수 있습니다.

당사의 기밀 의무, 직업상의 보안 유지 및/또는 서비스를 제공해야 하는 의무(예: 귀하의 고용주를 대상으로)로 인해 당사에서 귀하의 개인 데이터를 공개 또는 삭제하지 못하거나 귀하가 권리를 행사하지 못하게 되는 상황이 있을 수 있습니다. 상기 언급된 귀하의 권리는 당사에서 귀하의 개인 데이터를 처리하는 법적 근거에도 의거합니다.

당사의 개인 데이터 처리 방법과 관련하여 불만 사항이 있거나 추가 정보를 원하는 경우 언제든지 당사에 문의하십시오. 당사가 법적 권리를 보장하지 않는다고 느끼는 경우 귀하는 감독 기관에 불만 사항을 제기할 권리가 있습니다. 대부분의 경우 이 기관은 핀란드 데이터 보호 옴부즈맨(www.tietosuoja.fi)입니다.
