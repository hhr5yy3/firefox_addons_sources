Path to GIMP executable (e.g: C:\GIMP\GIMP.exe):

On Windows, if the path is not provided (is empty) "%ProgramFiles%\GIMP 2\bin\gimp-(version).exe" directory is used. On Linux, if the path is not set, it is assumed that the "gimp" binary can be found in the PATH environment variable. In Mac OS, If the box is empty open -a Gimp command is used, otherwise, the name of the provided program is used with the open command.
