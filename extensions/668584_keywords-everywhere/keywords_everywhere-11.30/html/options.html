Data Sources

Help

Keywords Everywhere primarily shows you data obtained from the Google Keyword Planner (GKP).

However, for some search queries such as banned terms (where you cannot run Google Ads on those terms) and mis-spellings, we use clickstream data.

If you always want to see only the data from GKP, change the default setting to only GKP.

Enable/Disable metrics

Help

You can choose which metrics you would like to see for the keywords. Changing this will affect which metrics the tool shows you everywhere.

Disabling these settings will not prevent credit usage. If you want to prevent credits from being used, either remove the API key from the above textbox or disable the settings in the 'Credit Usage For Widgets' section.

Help

You can highlight keywords based on specific values of the metrics.

For example you can highlight all keywords that have a search volume of more than 1000 per month.

Note that if you set up multiple conditions, then the keywords will be highlighted only if all the conditions are met.

For example if you set up highlights on volume > 1000 and CPC > $1, then only those keywords that have more than 1000 searches per month and have a CPC greater than $1 will be highlighted.

Supported websites

Help

This is a list of all the websites where Keywords Everywhere shows you data.

You can disable those websites where you do not want Keywords Everywhere to show you data, thereby saving on credit usage.

Credit usages

Help

On certain websites such as Google, YouTube & Bing, Keywords Everywhere places widgets that show you a list of keywords or trend charts. These widgets use up credits when they show you metrics.

These settings let you decide whether to see the search volume, CPC, competition & trend data in those widgets. Disabling them will stop the credits from being used up to show this data. It does not stop the widgets from being displayed.

Miscellaneos settings

Help

Show [Find Keywords] button: Adds a button at the top left of certain websites which provides the functionality of finding thousands of long-tail keywords for your search query using the Auto-suggest APIs from those websites

Show Difficulty Metrics: Adds a widget at the top right of every Google SERP and shows the following metrics - SEO Difficulty, On-Page Difficulty & Off-Page Difficulty

Show Trend Chart in Google/YouTube/Pinterest: Adds a widget at the top right of every Google SERP and shows the trend chart as well as historical search volume for the search term

Show YouTube Advanced Metrics: Enables the tool to automatically show the advanced metrics in YouTube Search Insights widget. Disabling this will show only basic metrics, and add a button for user to manually get the Advanced metrics

Show [Add All Keywords] button: Show button to add all keywords shown on a webpage to your Favorite Keywords list

Show [Export CSV] button: Show button to export all data shown on a webpage to a CSV file.

Show Metrics in Autocomplete Suggestion Drop Downs: Show volume metrics inside the auto-complete drop down on websites like Google & YouTube.

Show Chart in Google Trends: Show the chart in Google Trends with the search volume data enhancement.

Show Traffic Data in SERPs: View the URL & Domain traffic metrics embedded inside the search result pages.

Show Link Metrics in SERPs: View the Moz link metrics for the domains embedded inside the search result pages.

Show [Continue] Button on ChatGPT : You can disable the Continue floating button shown on ChatGPT.
