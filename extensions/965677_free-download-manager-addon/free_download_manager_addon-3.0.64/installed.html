FDM extension privacy

Free Download Manager does not collect and doesn't share any personally identifying information publicly or with third-parties.

When a browser starts a download, the Free Download Manager extension uses only the data that is necessary to download this file from this particular website. This data is necessary to start, or resume the download, or schedule this download in Free Download Manager. The data may include URL, internet address, cookies, query parameters, post data, namely all the data that the browser sends to the server when requesting the file.

This data is used on the local computer to start download only.

If you choose not to allow us to collect anonymous data, the FDM extension will be immediately removed from your browser.

Full details about the anonymous data we collect and what we do with it are provided in our Privacy Policy.
