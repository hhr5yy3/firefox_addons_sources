Version 1.99.1

  • Neue Titelsuche: Die Listensuche (Strg-Alt-Y) erfolgt nun direkt in der einfachen Suche.
    Nur zur Auswahl eines in der einfachen Suche nicht angebotenen Such-Index muss in die erweiterte Suche gewechselt werden.
    Die neue Titelsuche befindet sich d.Z. noch in Entwicklung bei Exlibris und muss extra aktiviert werden.     New Title Search The list search (Ctrl-Alt-Y) is now working directly in the basic search. Only when selecting a search index that is not available in the basic search is it necessary to switch to the advanced search.
    The new title search is currently still in development at ExLibris and needs to be activated separately.
  • ID-Links: Durch Regeln in den Einstellungen werden für alle passenden IDs der Form (ISIL/SN-Typ)IdIdIdIdIdId in $$0 und $$9 Links erzeugt,
    welche durch klicken auf die Feldnummer geöffnet werden.
    Im MDE nur mit Shift, Strg, Alt oder Meta. Je nach gedrückter Meta-Taste gehen die Links in einem neuen Tab oder Fenster auf.
    Alt öffnet einen neuen Tab. Shift ein neues Fenster. Strg einen neuen Tab, ohne auf diesen zu wechseln.
    Weitere Links gehen im gleichnamigen Tab/Fenster auf, unabhängig von der gedrückten Meta-Taste.
    Gegebenenfalls Tab vorher schließen, oder durch rausziehen zu einem eigenen Fenster machen.
    Die Tabs/Fenster sind nach der ISIL/SN-Typ benannt, d.h. dass z.B. alle GND-Links im gleichen Tab/Fenster aufgehen.
    Sind mehr als zwei Meta-Tasten (z.B: Shift+Alt) gedrückt werden unbenannte neue Tabs/Fenster geöffnet.     ID Links Through rules in the settings, for all matching IDs in the format (ISIL/SN-Type)IdIdIdIdIdId, links are created in $$0 and $$9,
    which can be opened by clicking on the tag number.
    In the MDE, this is only possible with Shift, Ctrl, Alt, or Meta keys. Depending on which Meta key is pressed, the links open in a new tab or window.
    Alt opens a new tab. Shift opens a new window. Ctrl opens a new tab without switching to it.
    Further links open in the tab/window with the same name, regardless of the pressed Meta key.
    If necessary, close the tab before or drag it out to make it a separate window.
    The tabs/windows are named after the ISIL/SN-Type. So all - e.g. gnd - links will be opened in the same tab.
    If more than two Meta (e.g. Shift+Alt) keys are pressed, unnamed new tabs/windows will be opened.
  • URL-Links: Für alle URLs werden ebenfalls Links wie für die IDs erzeugt. Auch in nicht von EL vorgesehene (lokale) Felder. URL Links Links are also generated for all URLs, similar to IDs. This also applies to fields that are not specified by EL (local) fields.
