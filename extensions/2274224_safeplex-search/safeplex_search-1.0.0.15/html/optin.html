Privacy Notice

We need the following permissions to enable full functionality:

  • Permission to collect and process the following user data:

    IP Address: This is sent to our server to determine if a visited domain has a potential recognized threat. We do not use this for any kind of targeting.

    Permission to collect and process the following anonymous information:

    Anonymized Domain Information: This is sent to our server to process your visited domains through our security filters to determine if they are safe to browse. We do not store or share this anonymized data with anyone.

    Extension Name, Extension Version, Extension Category, Extension Family, Application ID: This data is collected to make improvements to our product and help us in troubleshooting issues and tracking outages.

    ** If you choose not to provide us with this data, it would impact our ability to provide the core functionality of this product. You can still continue using the product, but we won’t be able to identify if the URLs visited by you are secure and won’t be able to warn you about any unsafe URLs.
