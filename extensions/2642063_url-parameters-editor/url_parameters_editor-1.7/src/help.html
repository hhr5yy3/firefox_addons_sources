Browser parameters editor - Help Page

Welcome to the help page of Browser parameters editor

The usage of this extension should be pretty straightforward, this help page higlight the its features.

What you can do with this extension

Browser parameters editor is an extension that let users to see all url parameters (the url part after the ?) in a easy to read wayy. You just need to click on the extension icon, and a popup opens with all the parameters listed opens. From there you have several options:

  • Edit existing parameters
  • Add a new parameter
  • Remove an existing parameter
  • Save the list in different format (JSON, xml, csv)
  • Import the parameters list from a CSV file (more format will be added in future, be aware that this feature could not work on firefox)
If you have changed anything in the parameter list, you just need to press the update button to reload the page with the your changes. Attention: To load your changes don't press the browser reload button, but only the extension Update button

Export Parameters

Current export format supported are:
  • CSV
  • XML
  • Json
To export the current parameters into a file Json, CSV or XML press the export button and then choose the format, you will be prompted with a Download file dialog, from here you can either Save it locally or just open it with your preferred editor.

Import parameters

Currently for the import only one format is supported: CSV. Press the import button, a "File Open" dialog appear, select your CSV file with the parameters list and press open. The page should be automatically refreshed with the parameters from the CSV file Attention: This feature is currently working only on Chrome, unfortunately on firefox is broken, hopefully this will be fixed in the near future, if possible
