1. Video Downloader Prime uses the default browser's download location. To change the default download location, please head to settings page and change the default location.
2. Trigger download if video size is more than Bytes.

Default value is 307200 Bytes.

3. Check this box to show the resolution for each video item in the toolbar-popup UI

This option will show the resolution for each captured video. (disabled by default)

Note: in some occasions, enabling this option may cause the browser to crash. So, please proceed with caution when this option is active.

4. Open support page whenever the addon is upgraded

Uncheck this box to NOT open the support page on every update. (default is to open)

5. Use webpage's exact address (tab URL) to list all captured Videos

When this option is checked, the addon uses the exact URL for any webpage to list all captured Videos in the toolbar-popup UI. (default is NOT active)

If you disable this option, the addon uses webpage DOMAIN to list all captured Videos.

In other words, once you use page URL, the addon lists the Videos for that exact URL only.

But using page DOMAIN, the addon lists all the Videos currently captured for the whole domain name (the video list may become long).
