Notifications

ReloadMatic can notify you of events that could be of interest to you. You can set which notifications you wish to receive here.



Smart Timing

This feature can be enabled or disabled for each tab separately, and if enabled, adjusts a pending reload if there is user activity inside it. This way it avoids reloading the page while you are interacting with it. Its behavior can be influenced with the following options.

Pinning

If you enable this option, pinning or unpinning a tab will automatically set or clear the tab's "Remember page" property. Note: Only takes effect on newly pinned/unpinned tabs.

URL Fixing

A fixed URL is often used together with Sticky Reload, because one common use-case for a fixed URL is for pages that redirect. In those cases you'll probably want ReloadMatic to not disable the reload timer upon the redirect, which is what Sticky Reload is for. This option allows you to automatically set Sticky Reload when setting a fixed URL. Note: Only takes effect on newly set/unset fixed URLs.

Remembered Pages

Here you can delete all remembered pages. Saved ReloadMatic settings for remembered URLs are then deleted and all newly loaded pages will start with the default options unless changed and remembered again.
