TITLE
The Title Tag is found in the <head> section of a page and specifies its title. It also serves as a clickable link on the search engine results page and as a name shown in the browser tab.

DESCRIPTION
The Meta Description Tag is found in the <head> section of a page and specifies its description. It serves as a page description on the search engine results page (but can often be re-written by Google).

URL
This is the URL of the page you are researching right now. A URL is a unique web address of a page.
CANONICAL

The Canonical Tag is found in the <head> section of a page. The Canonical Tag tells search engines which URL should be seen as the master URL. It helps to avoid content duplication.

META ROBOTS
The Meta Robots tag gives search engine bots instructions on indexing your page. ‘Index, follow’ is a default state.
