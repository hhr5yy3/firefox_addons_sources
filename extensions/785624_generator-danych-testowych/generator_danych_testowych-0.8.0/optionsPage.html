Pokaż dostępne opcje dla danych użytkownika

%randomNumber:n% - generuje losowy ciąg cyfr o podanej długości (np. n = 5)

%randomString:n% - generuje losowy ciąg liter o podanej długości (np. n = 5)

%randomNumberString:n% - generuje losowy ciąg cyfr i liter o podanej długości (np. n = 5)

%randomHexadecimal:n% - generuje losową wartość w systemie szesnastkowym o podanej długości (np. n = 5)

%timestamp% - wstawia aktualny znacznik czasowy

%randomFirstname% - wstawia losowe imię

%maleFirstname% - wstawia losowe imię męskie

%femaleFirstname% - wstawia losowe imię żeńskie

%randomSurname% - wstawia losowe nazwisko

%maleSurname% - wstawia losowe nazwisko męskie

%femaleSurname% - wstawia losowe nazwisko żeńskie

%city% - wstawia losowe miasto

%street% - wstawia losową ulicę
