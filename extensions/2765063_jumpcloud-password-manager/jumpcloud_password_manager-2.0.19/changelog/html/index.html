2.0.0 Feb 01, 2024

  • This release contains a new design for the browser extension. It retains full functionality but has a new look and feel and has been streamlined with the user in mind. These updates include:
    • an updated design that is easier to navigate between items, folders, and settings
    • an expanded and specific settings menu
    • item-specific settings
    • enhanced sorting options
  • When adding passwords from a website the password manager extension is easier to use, and includes the ability to ignore to quickly dismiss the prompt.
    • The password can be assigned to a folder straight from this prompt.
    • Users can also exclude any website from the browser extension when logging into a website or from the settings menu of the browser extension.
  • A password generator is available in the top toolbar of the browser extension. Using a generated password greatly increases a user's security posture.
  • With this release the workflow for editing items and moving items between folders has been simplified.
