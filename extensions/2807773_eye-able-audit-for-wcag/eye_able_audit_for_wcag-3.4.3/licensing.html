Wenn diese Option deaktiviert ist, werden die Markierungen auf der Webseite beim Öffnen des Dashboards nicht mehr angezeigt.

Running a test can require a lot of performance. Disabling this setting will run test only when entering the on site dashboard.

Mit dem aktiven Screenreader Fokus Modus wird beim Klick auf den Highlight-Button der Fokus direkt auf das Element in der Seite gesetzt. Dadurch kann der Fehler mit dem Screenreader genauer untersucht werden. Zurück zur Eye-Able Audit Toolbar kommt man mit ALT + F7.

Aktivieren Sie diese Funktion nur nach einem Hinweis des Eye-Able Supports. In sehr seltenen Fällen kann das Alfa Prüfmodul fehlschlagen und kann dann deaktiviert werden. Der Einfluss auf das Prüfergebnis ist minimal.
