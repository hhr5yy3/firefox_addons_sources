F-Secure 浏览保护隐私策略

2022 年 9 月

简要说明

F-Secure 浏览保护是通过 F-Secure 安全产品保护加密浏览的扩展。

F-Secure 的安全产品需要浏览器扩展来保护您的网页浏览、网上银行和购物，并在您浏览互联网时向您显示安全信息。

总之，就 F-Secure 浏览保护和隐私而言：
  • 我们收集匿名安全数据来保护您和您的设备；
  • 数据收集的重点不在于您，而在于您的设备和我们的服务。

详细说明

术语

  • F-Secure 浏览保护是此隐私策略适用的 Firefox 浏览器扩展。
  • F-Secure 安全产品是指由 F-Secure Corporation 提供的任何与 F-Secure 浏览保护扩展一起使用的安全产品。该等产品可能带有 “F-Secure” 以外的品牌名称，在这种情况下，可通过 "by F-Secure" 徽标来识别。
  • F-Secure Security Cloud 是由 F-Secure 运营的基于云端的网络威胁分析系统。通过 Security Cloud，F-Secure 可了解最新全球威胁概况，并在发现新威胁的那一刻起保护我们的客户。

通过 F-Secure 浏览保护收集的数据

我们检查您打开的网页的 URL（网页地址）、从您打开的网页引用的 URL，并跟踪您打开和关闭的浏览器选项卡。该信息将传递到您在计算机上安装的 F-Secure 安全产品。安全产品向 F-Secure Security Cloud 发送有关 URL 的查询，以获得 URL 的安全等级；根据获得的安全等级，F-Secure 浏览保护会阻止访问被评为有害、可疑、非法或（如果安全产品如此配置）不适合儿童的网页。跟踪打开和关闭的浏览器选项卡仅限于您的计算机，以便跟踪您的安全银行会话；我们不会将有关您的浏览器选项卡的信息转发给 F-Secure Security Cloud。

如果安装的安全产品是这样配置的，则 F-Secure 浏览保护会修改互联网搜索查询（例如，使用 Google 或 Bing）以强制搜索以“安全搜索”模式运行，阻止色情、暴力等露骨内容，并在搜索结果中忽略。

当您使用 Google、Bing 和其他搜索引擎搜索互联网时，我们会检查您的搜索结果的 URL，并如上所述从 F-Secure Security Cloud 查询其安全评级。然后，我们会通过在您的搜索结果旁边插入“评级图标”来修改搜索结果页面，以便您查看搜索结果中的哪些链接是安全的、有害的、可疑的、非法的或不适合儿童的。

F-Secure 浏览保护仅监控和收集正在打开的 URL 以及从您打开的网页中引用的 URL。我们不会读取或修改您打开的网页内容的任何其他部分，但其中包含的 URL 和用于插入上述“评级图标”的搜索结果除外。

您打开的 URL 或您打开的网页中包含的 URL 会匿名转发到 F-Secure Security Cloud（用于上述目的），不会有任何信息让我们识别到您是打开网页的一方。

您的权利

您对于我们持有的关于您的数据享有权利。尤其是，您对于我们持有的关于您的个人数据享有以下权利：

  • 访问和改正。您有权询问我们持有关于您的哪些个人数据，在此情况下，您还可获得一份我们可确认与您有关的数据。如果您发现该等数据中有任何错误（如过时信息），则我们会让您联系我们的客户服务部门以解决该问题。我们有些服务门户网站可供您更新您的客户信息。因此，您应更新您的个人数据的任何更改，如地址或电子邮件地址的更改。如果您自己无法更新，则可通知我们必要的更改。
  • 异议。您有权对特定的个人数据处理提出异议，例如包括处理个人数据用于营销目的，或基于合法权益处理您的个人数据。在后一种情况下，您需要针对您的异议确定法律上有效的基本原理。
  • 被遗忘的权利。您还有权请求我们停止存储您的个人数据并将其删除。在此情况下，您需要针对您的请求确定法律上有效的基本原理。
  • 可移植性 。您还有权询问您自己提供的个人数据 - 根据合同或您的同意书。您可请求结构化、常用或机器可读格式的数据，而且，若技术上可行，数据还可传输至其他控制器。
  • 撤销同意书。如果根据您的同意书处理个人数据，则您有权随时通过相应设置撤销您的同意书。就可识别的产品分析数据而言，您可在服务用户界面找到设置。您还有权通过偏好中心（通过链接访问）退出我们的营销通讯。
  • 限制。如果您确定我们持有的关于您的数据不正确，或我们无权合法使用，则可请求我们停止进一步处理您的个人数据，仅仅将其存储，直至问题解决。

您可通过我们的客户服务职能行使自身权利。我们的联系链接位于“联系信息”部分。

请注意，我们的保密义务、职业机密权利和/或我们（向您的雇主）提供服务的义务可能会禁止我们披露或删除您的个人数据，或阻止您行使您的权利。您的上述权利也依赖于我们处理您个人数据所采用的法律依据。

若您对我们处理您个人数据的方式有意见，或需要进一步了解相关信息，请随时与我们联系。若您认为我们没有为您赋予您的法定权利，您有权向监管机构发起投诉。大多数情况下，该机构为芬兰数据保护监察专员 (Finnish Data Protection Ombudsman, www.tietosuoja.fi)。
