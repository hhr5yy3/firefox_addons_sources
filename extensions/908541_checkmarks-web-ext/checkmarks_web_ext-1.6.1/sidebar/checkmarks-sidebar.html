Symbols

play_circle_outline Starts or resumes loading bookmarks and favicons.
pause_circle_outline Pauses a run and stores results.
highlight_off Cancels the current run or discards a paused run.
help_outline Shows this help dialog.
settings Opens the preferences page.
delete Deletes a bookmark permanently (after confirmation).
check_circle Removes an entry from list but does not delete the bookmark (false negative).
launch Launches a bookmark in a new tab.
folder_open Hover the icon to display the directory-path of the bookmark!
save Updates the bookmark with the url of the tab you've previously opened with the launch-action launch (appears after launch, see above).


Hints

  • Adjust the width of your sidebar to better see longer bookmark-titles. The list of errors can be scrolled horizontally.
  • Symbols in the result list are explained in a tooltip when you hover the icon.
  • Filters are only applied to the visible list. Internally all errors are stored and can be filtered or displayed at any time.
warning

Warning

Clicking the trash icon permanently removes the corresponding entry from your bookmarks.

After the confirmation of the first removal this warning will not appear again.
