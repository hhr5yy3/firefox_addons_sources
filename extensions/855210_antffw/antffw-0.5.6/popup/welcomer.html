Welcome to Ant Download Manager extension!

The extension helps to download any content from the Internet: exe, zip, doc, mp4, avi, mp3, etc...
Supports network HTTP(S), FTP(S) and media stream M3U8, MPEG-DASH, VIMEO-STREAM protocols.

WARNING: Our extension works in cooperation with Ant Download Manager (AntDM) software. If you want to use this extension, you should download AntDM and install it on your computer. Do not forget to turn on AntFF2 extension for Firefox Mozilla browser.
