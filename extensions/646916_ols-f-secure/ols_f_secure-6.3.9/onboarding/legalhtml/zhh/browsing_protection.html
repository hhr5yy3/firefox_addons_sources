F-Secure 瀏覽保護私隱權原則

2022 年 9 月

簡要說明

F-Secure 瀏覽保護是透過 F-Secure 安全產品保護加密瀏覽的延伸。

F-Secure 安全產品需要瀏覽器延伸來保護您的網頁瀏覽、線上銀行和購物，並在您瀏覽互聯網時向您顯示安全資訊。

總的來說，就 F-Secure 瀏覽保護和隱私而言：
  • 我們收集匿名安全資料以保護您和您的裝置；
  • 資料收集的重點不在於您個人，而在於您的裝置和我們的服務。

詳細說明

術語

  • F-Secure 瀏覽保護是此私隱權原則適用的 Firefox 瀏覽器延伸。
  • F-Secure 安全產品是指由 F-Secure Corporation 提供的任何與 F-Secure 瀏覽保護延伸一起使用的安全產品。該等產品可能帶有「F-Secure」以外的品牌名稱，在這種情況下，可透過「by F-Secure」標誌來識別。
  • F-Secure Security Cloud 是由 F-Secure 運營的基於雲端的網路威脅分析系統。透過 Security Cloud，F-Secure 可了解最新全球威脅概況，並在發現新威脅的那一刻起保護我們的客戶。

透過 F-Secure 瀏覽保護收集的資料

我們檢查您打開的網頁的 URL (網頁地址)、從您打開的網頁引用的 URL，並跟踪您打開和關閉的瀏覽器索引標籤。該資訊將傳遞至您在電腦上安裝的 F-Secure 安全產品。安全產品向 F-Secure Security Cloud 傳送有關 URL 的查詢，以獲得 URL 的安全等級；根據獲得的安全等級，F-Secure 瀏覽保護會阻止存取被評為有害、可疑、非法或 (如果安全產品如此設定) 不適合兒童的網頁。跟踪打開和關閉的瀏覽器索引標籤僅限於您的電腦，以便跟踪您的安全銀行工作階段；我們不會將有關您的瀏覽器索引標籤的資訊轉寄給 F-Secure Security Cloud。

如果安裝的安全產品是這樣設定的，則 F-Secure 瀏覽保護會修改網路搜尋查詢 (例如，使用 Google 或 Bing) 以強制搜尋以「安全搜尋」模式執行，阻止色情、暴力等露骨內容，並在搜尋結果中忽略。

當您使用 Google、Bing 和其他搜尋引擎搜尋網路時，我們會檢查您的搜尋結果的 URL，並如上所述從 F-Secure Security Cloud 查詢其安全評級。然後，我們會透過在您的搜尋結果旁邊插入「評級圖示」來修改搜尋結果頁面，以便您查看搜尋結果中的哪些連結是安全的、有害的、可疑的、非法的或不適合兒童的。

F-Secure 瀏覽保護僅監控和收集正在打開的 URL 以及從您打開的網頁中引用的 URL。我們不會讀取或修改您打開的網頁內容的任何其他部分，但其中包含的 URL 和用於插入上述「評級圖示」的搜尋結果除外。

您打開的 URL 或您打開的網頁中包含的 URL 會匿名轉寄至 F-Secure Security Cloud (用於上述目的) ，不會有任何資訊讓我們識別到您是打開網頁的一方。

您的權利

您對於我們持有的關於您的資料享有權利。尤其是，您對於我們持有的關於您的個人資料享有以下權利：

  • 存取和改正。您有權詢問我們持有關於您的哪些個資料，在此情況下，您還可獲得一份我們可確認與您有關的資料。如果您發現該等資料中有任何錯誤 (如過時資訊)，則我們會讓您聯絡我們的客戶服務部門以解決該問題。我們有些服務入口網站可供您更新您的客戶資訊。因此，您應更新您的個人資料的任何變更，如地址或電子郵件位址的變更。如果您自己無法更新，則可通知我們必要的變更。
  • 異議。您有權對特定的個人資料處理提出異議，例如包括處理個人資料用於行銷目的，或基於合法權益處理您的個人資料。在後一種情況下，您需要針對您的異議確定法律上有效的基本原理。
  • 被遺忘的權利。您還有權請求我們停止存儲您的個人資料並將其刪除。在此情況下，您需要針對您的請求確定法律上有效的基本原理。
  • 可移植性。您還有權詢問您自己提供的個人資料 - 根據合約或您的同意書。您可請求結構化、常用或機器可讀格式的資料，而且，若技術上可行，資料還可傳輸至其他控制器。
  • 撤銷同意書。如果根據您的同意書處理個人資料，則您有權隨時透過相應設定撤銷您的同意書。就可識別的產品分析資料而言，您可在服務用戶界面找到設定。您還有權透過偏好中心 (透過連結存取) 退出我們的行銷通訊。
  • 限制。如果您確定我們持有的關於您的資料不正確，或我們無權合法使用，則可請求我們停止進一步處理您的個人資料，僅僅將其存儲，直至問題解決。

您可透過我們的客戶服務職能行使自身權利。我們的聯絡連結位於「聯絡資訊」部分。

請注意，我們的保密義務、職業機密權利和/或我們（向您的雇主）提供服務的義務可能會禁止我們披露或刪除您的個人資料，或阻止您行使您的權利。您的上述權利也依賴於我們處理您個人資料所採用的法律依據。

若您對我們處理您個人資料的方式有意見，或需要進一步瞭解相關資訊，請隨時與我們聯絡。若您認為我們沒有為您賦予您的法定權利，您有權向監管機構發起投訴。大多數情況下，該機構為芬蘭資料保護監察專員 (Finnish Data Protection Ombudsman, www.tietosuoja.fi)。
