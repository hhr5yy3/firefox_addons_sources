YandexMusic.pro - скачивай музыку быстро, удобно и бесплатно!


Обязательно прочитайте правила использования расширения

  • Расширение позволяет скачивать аудиотреки и целые альбомы с сайта Яндекс.Музыка.
  • По правилам Яндекс Музыки для прослушивания и скачивания музыки требуется войти в аккаунт Яндекса.
    Без входа все треки скачиваются по 30 секунд.
  • По умолчанию аудио треки скачиваются с битрейтом 192 kbps.
    При наличии подписки «Плюс» треки скачиваются с битрейтом 320 kbps.
  • Расширение скачивает файлы со всеми метаданными (обложка, название, артисты, альбом, год выпуска и т.д.) только в том случае если они имеются на сервере.
  • Расширение предоставляет возможность массового скачивания треков.
    Для скачивания одновременно всех треков необходимо прокрутить страницу с аудио в самый низ.
    Максимальное кол-во скачиваемых треков при нажатие на кнопку «Скачать все аудиозаписи» равно 100!
    Разделяйте на отдельные альбомы если у вас в подборке больше 100 аудиозаписей.
  • При первом запуске для включения меню расширения в верхнем углу вашего браузера нажмите на значок
    «Пазл» → «Шестеренка» → «Закрепить расширение».
  • Дополнительные опции доступны через меню настроек
    (правый клик мыши по иконке → Управление расширением → Настройки).
  • В расширение можно указать собственную папку для скачивания аудиозаписей.
    По умолчанию, все аудиозаписи скачиваются в папку, указанную для загрузок всех файлов в вашем браузере.
  • Если в вашем браузере установлена опция «Всегда указывать место для скачивания» - браузер будет показывать соответствующее окно вне зависимости от того указана папка или нет.
  • Расширение работает корректно только в браузере Mozilla Firefox.
  • Для корректной работы расширения отключите все другие приложения для скачивания музыки, видео и т.д.
    Иначе названия скачиваемых аудиозаписей могут быть некорректными.
  • Жалобы по всем вышеуказанным пунктам не принимаются.

Продолжая использовать расширение Вы соглашаетесь со всеми вышеуказанными пунктами.
