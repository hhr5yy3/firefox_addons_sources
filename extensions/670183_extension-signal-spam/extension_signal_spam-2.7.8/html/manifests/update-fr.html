Signaler, c'est agir

Version 2.7.8:

  • Mise à jour pour la dernière version du webmail AOL

Signaler un e-mail ou site illégitime :

Avec cette extension, vous pouvez signaler un e-mail, indésirable ou fraduleux, depuis un des webmails compatibles (Gmail, Outlook.com, Yahoo! Mail, Orange, Free, SFR, OVH, LaPoste.net), ou signaler un site web frauduleux en cliquant sur le bouton Signal Spam installé dans la barre d'outils de votre navigateur.

Pour signaler un e-mail : Ouvrez un e-mail ou sélectionnez les e-mails à signaler puis cliquez sur le bouton Signal Spam dans la barre d'outils de votre navigateur.

Pour signaler une page web frauduleuse (Phishing): Ouvrez la page web dans votre navigateur, puis cliquez sur le bouton Signal Spam dans la barre d'outils de votre navigateur.

Grâce à vos signalements, vous pouvez contribuer à protéger l'ensemble des utilisateurs Signal Spam.

Protection contre les sites frauduleux :

L'extension Signal Spam vous alertera, dans votre webmail, si vous ouvrez un e-mail contenant un lien vers un site frauduleux. Une alerte sera également affichée si vous consultez un site web répertorié comme frauduleux.
Ces alertes ne s'appliquent qu'aux liens vers des sites web déjà répertoriés comme frauduleux.
