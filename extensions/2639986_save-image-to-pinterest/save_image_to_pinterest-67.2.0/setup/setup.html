Open With Photoshop Setup

Open With Photoshop is a new companion for web and graphic designers to open up any web image with Adobe Photoshop via a single and quick mouse click. This is the first time you are using Open With Photoshop. Please follow below steps to properly setup the extension to be able to use it.

1. Download OWP Communicator

You have successfully installed Open With Photoshop add-on but you just need some easy steps to set things up before using Open With Photoshop. Actually we create software which works out of box that does not require any additional step to use. Open With Photoshop for Firefox works without any configuration and you can use it as soon as you install the add-on. But Google Chrome restricts extensions to access other programs installed on your computer for security reasons. Therefore you need to have a communicator software which needs to carry messages between Open With Photoshop extension and Photoshop itself. We are really sorry about this extra requirement for Google Chrome version of Open With Photoshop, but this requirement is a necessity beyond our control. Anyway, required steps are not complicated and it can be done in only 15 seconds, fortunately.

1. Please download our communicator software to your computer by clicking the below download link.

Download OWP Communicator

2. Extract the archive file for a secure place in your computer.

3. Double click on the OWP.Communicator.exe.

4. If everything is OK, you will get a Success notification box.

1. Please download our communicator software to your computer by clicking the below download link.

Download OWP Communicator

2. Extract the downloaded archive file and drag extracted OWP Communicator into Mac OS X's Applications folder.

3. After you transfer the file into Applications folder, double click on the OWP Communicator to initialise the application for the first time.

1. Please download our communicator software to your computer by clicking the below download link. Please note that only Ubuntu Desktop with Unity is tested and supported for Linux, as of now.

Download OWP Communicator

2. Extract the downloaded archive file. After extraction, cd into the extracted directory by Terminal.

3. Run ./OWP.Communicator command in Terminal. Type your root password for administrative privileges. In a second or so, you will be welcomed by a success installation notification.
