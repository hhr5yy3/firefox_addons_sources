Version 4.15
Released September 8, 2022

  • Installer: FFmpeg updated to the latest version

  • Bugfix: scoring setting to check non-media content types works incorrectly
  • Bugfix: scoring settings to not detect videos for which server provided 0 bytes size works incorrectly
  • Bugfix: when video detection is turned off and browser restarted, the extension continues to detect videos on some websites
  • Bugfix: when Native app is installed, scoring videos are prompted to download with the browser, instead of Native app

Version 4.14
Released July 29, 2022
  • Deep memory and storage optimizations — detected videos are no longer stored in browser's storage
  • Refactored UI, made more compact and fast

  • Page ALL VIDEOS renamed back to DOWNLOADS
  • New settings for UI, MPD, Scoring
  • New feature: mute/unmute video detection on tab
  • New feature: enable/disable video detection and separate protocols
  • New feature: Show properties on downloaded video file
  • New feature: Move to Recycle bin the downloaded file

Version 4.10
Released Dec 20, 2021
  • New YouTube protocol. Ability to download videos without Native app, ability to convert videos to MP3 format
  • The existing YouTube protocol is now called YouTube Now Playing (YTNP)
  • New experimental MPEG-DASH (MPD) protocol (may not download some videos)
  • Settings feature
  • New UI (popup made compact, new internal pages)
  • Numerous bugfixes, scoring rules updated
Version 4.9

Version 4.8
Released Jan 27, 2021
  • Improved general quality of video detection and downloads
  • Added support of the Waterfox web browser
  • Improved error handling in MPD protocol. The new version of MPD protocol is scheduled for version 4.9
  • Bugfix: when downloading video in private mode, popup displays the old video title instead of the new filename
  • Button TRY VIDEO SEARCH replaced with TOP VIDEO SITES
  • Added new rules for suppressing video segments, previews, and advertisement

Version 4.7
Released Apr 21, 2020
  • Added support of First Party isolation (FPI). AVD can now download videos when FPI is on, as well as using the Tor browser
  • Scoring system updates: detection of segments and previews suppressed, fixed video downloads
  • HLS protocol: full path to playlist added to configuration dialog. Helps to distinguish different qualities when media playlists are all have the same filename
  • Native app: fixed bug in HLS protocol when demuxer.txt file was not deleted
  • Native app: when downloading videos in Private mode, the previously used folder is remembered and proposed for the next download
  • Native app: FFmpeg version updated to 4.2.2 (helps with some Vimeo videos)

  • Scoring system: Google/YouTube videogoodput file detection suppressed

Version 4.5
Released Feb 11, 2020

  • New feature: UI: added buttons for opening temp files directory (top level and this session)
